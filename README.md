# Semmed Label System

## Settings

1. Install Package
    ```
    npm install
    ```

2. Compile
    ```
    # For Debugging
    npm run watch

    # For Development
    npm run build

    # For production
    npm run deploy
    ```

3. Run
    ```
    # Run website
    node app.js
    
    # Run API server
    npx babel-node server/index.js
    ```
    And open http://localhost/9999 in browser
