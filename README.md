# Medical Relation Labeling System

## Settings

1. Install Package
    ```
    npm install
    ```

2. Compile
    ```
    # For Debugging
    npm run watch

    # For Development
    npm run build
    ```

3. Run
    ```
    # Run API server
    npm run api

    # Run website
    npm start
    ```
    And open http://localhost/9999 in browser
